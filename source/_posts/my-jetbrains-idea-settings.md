* Keymap
  * Scheme: `Visual Studio copy`
    * Main menu
      * Window
        * Editor Tabs
          * Split Vertically: `Ctrl + \`
          * Move Right: `Alt + \`
    * Tool Windows
      * Database: `Alt + D`
      * Terminal: `Alt + Grave accent`
    * Other
      * New...: `Ctrl + N`
      * Show in Explorer: `Alt + Shift + R`
* Editor
  * General
    * Mouse
      * [x] Change font size (Zoom) with Ctrl + Mouse Wheel
  * Font
    * Font: `Consolas`
    * Size: `16`
    * Line spacing: `1.7`
  * Code Style
    * Hard wrap at `500` columns
